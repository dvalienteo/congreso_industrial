⚙ Configuración — Google Sheets + Apps Script

Usa la misma URL del Web App que ya tienes desplegado en code_patrocinador.gs. No necesitas crear otro script — tesorería y el portal del patrocinador comparten el mismo script y hoja.

1️⃣ Pegar URL del Web App

Ve a Google Apps Script → Implementar → Administrar implementaciones → copia la URL del Web App.

✓ URL guardada. También pégala como valor de SHEETS_URL_DEFAULT en el HTML para que funcione en todos los dispositivos.

2️⃣ Columnas en Google Sheets (fila 1)

A: No. | B: Recibo | C: Fecha | D: Lugar | E: Patrocinador | F: Monto Q | G: En letras | H: Plan | I: Observaciones | J: Timestamp

3️⃣ Permisos del Web App

Al desplegar: Ejecutar como: Yo · Acceso: Cualquier persona. Si cambias el código, crea una nueva implementación y actualiza la URL aquí.

⚠ Resetear Sistema

Eliminará TODOS los recibos del caché local y de Google Sheets. Irreversible.
